# Yojana_Guide
----------------------------------------------------------------------------------------------------------------------------------------------------------------------
A Llama 2 based AI Chatbot System for accessing Indian Governmnet Schemes
This is a Repository containing a Team Project:Yojana Guide: A Chatbot System for Accessing Indian Government Schemes
The Chatbot is Made using
Llama 2 chat 7b AWQ (quantised) model, Government schemes are web scraped from the official government schemes website using Beautiful Soup.
For training the model it requires GPU based system (T4 GPU 2*16 GB RAM or any other) on GOOGLE COLAB.
Attached Files are of CODE and DATASETS, since model can be downloaded from hugging face's official website.
